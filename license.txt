MODIFICATIONS

EN
The data-csv folder has been the only folder that has been used on this Project.  The following modifications have been made:
- A ages.csv has been added with the respective time periods;
- A player_image_links.csv has been added, cloning the players.csv names and adding the wikipedia-image-url for the players that have one;
- A players_no_image.csv has been added, cloning the players names and adding a "no_image_on_page" for the players that don't have one;
- The teams.csv have the following columns added: flag, world cups, ranking and team_name_pt. The flag column uses the https://flacdn.com website for the respective flag icons;
- The tournaments.csv have 3 added columns: tournament_name_pt, host_pt and winner_pt. Those are the same as the one the database already have, but in brazillian portuguese.

PT-BR
A pasta "data-csv" é a única que foi utilizada nesse projeto. As seguintes modificações foram feitas:
- O "ages.csv" foi adicionado com os períodos de tempo;
- O "player_image_links.csv" foi adicionado, clonando os nomes do "players.csv" e adicionado a coluna "wikipedia-image-url" para os jogadores que têm uma imagem;
- O "player_no_image.csv" foi adicionado, clonando os nomes dos jogadores e adicionado uma coluna "no_image_on_page" para os jogadores que não têm uma imagem;
- Foram adicionadas as seguintes colunas no "teams.csv": "flag", "world cups", "ranking" and "team_name_pt". Elas são os mesmos dados que a database já tem, mas em português.
- Foram adicionadas 3 colunas no "tournaments.csv": "tournament_name_pt", "host_pt" e "winner_pt". Elas são os mesmos dados que a database já tem, mas em português.
